Fun Telegram Bot Documentation

This document is for the Funttastic Bot integrated on Telegram. The bot is capable of Market Making (MMing) using Kujira’s network (testnet and mainnet) and Hummingbot Gateway (HB Gateway).

A tutorial on how to set up your own bot is available here.

A Supervisor is an entity responsible to supervise all workers configured, collecting statistics globally.

The Worker is an entity to work in a specific market with a specific strategy.
