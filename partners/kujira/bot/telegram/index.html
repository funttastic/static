Fun Telegram Bot Documentation

This document is a documentation for the Funttastic Bot integrated on Telegram. The bot is able to do Market Making (MMing) using Kujira’s network (testnet and mainnet) and Hummingbot Gateway (HB Gateway).

A tutorial on how to setup your own bot is available here.

Supervisor

Here is the documentation explaining all aspects of the Supervisor’s summary on Telegram:

  • Strategy name: Strategy version: Strategy ID:supervisor
  • Supervisor ID
  • Workers
  • Allowed workers
  • Active workers
  • Stopped workers
  • Profit and Loss (in US Dollars)
  • Global or combined (from all workers) profit and loss
  • Global percentual profit and loss
  • Global profit and loss amount
  • Wallets (in US Dollars)
  • Wallet initial value
  • Wallet previous value
  • Wallet current value
  • Percentual current to initial wallet value change
  • Percentual current to previous wallet value change
  • Balances (in US Dollars)
  • Total free/not used balance
  • Total balance amount locked in orders
  • Total unsettled balance amount (not free and not in orders)
  • Total balance (free + locked/orders + unsettled)
  • Settings
  • Amount of time the worker waits to run again
  • Defines if the worker has to run only once or not (true/false)

Worker

Here is the documentation explaining all aspects of the Worker’s summary on Telegram:

  • Strategy name: Strategy version: Strategy ID:worker:Worker ID
  • Worker ID
  • Network
  • Market
  • Wallet ID
  • Profit and Loss (in US Dollars)
  • Percentage
  • Amount
  • Balances (in US Dollars)
  • Total (base + quote)
  • Total free/not used balance
  • Total balance amount locked in orders
  • Total unsettled balance amount (not free and not in orders)
  • Total balance (free + locked/orders + unsettled)
  • Tokens
  • Base token of the current market
  • Current base token price
  • Base token free/not used balance
  • Base token balance locked in sell orders
  • Base token unsettled balance (not free and not in order)
  • Base token total balance
  • Quote token of the current market
  • Current quote token price
  • Current quote token free/not used balance
  • Current quote token balance locked in sell orders
  • Current quote unsettled token (not free and not in order)
  • Current quote token total balance
  • Wallet (in US Dollars)
  • Wallet initial value
  • Wallet previous value
  • Wallet current value
  • Percentual current to initial wallet value change
  • Percentual current to previous wallet value change
  • Price
  • Price effectively used to place orders
  • Current price from the market or ticker (mainnet price)
  • Orders
  • Quantity of new orders created in this round
  • Quantity of new orders canceled in this round
  • Fees
  • Name of the native token
  • Total amount of fees paid when creating orders
  • Total amount of fees paid when canceling orders
  • Total amount of fees paid while withdrawing or settling the balances
  • List of newly created orders
  • List of currently untracked orders
  • List of orders canceled
  • List of all filled orders
  • Settings
  • Amount of time the worker waits to run again
  • Type of the order (limit or market)
  • Strategy to define the current market price (middle, ticker, last fill): SAP (simple average price), WAP (weighted average price), VWAP (volume weighted average price)
